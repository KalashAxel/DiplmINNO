# Проект: вариант В (кредитный скоринг)

## Что внутри
- notebooks/ — 4 блокнота с кодом (подготовка данных, базовая модель, эксперименты, итоговая оценка).
- sql/ — schema.sql, layers.sql, queries.sql для PGSQL.
- src/utils.py — вспомогательные функции (метрики, порог, недискриминационность, скорость).

## Как запустить
1) Положить CSV в папку data/ как `GiveMeSomeCredit-training.csv`.
2) Открыть Jupyter из корня проекта.
3) Запускай блокноты по порядку.

## PGSQL
- Выполнить sql/schema.sql
- Импортировать CSV в raw.credit (через COPY/pgAdmin)
- Выполнить sql/layers.sql
- Использовать sql/queries.sql для контроля качества.
